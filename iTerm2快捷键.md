## 标签

- 新建标签：⌘ + t
- 关闭标签：⌘ + w
- 切换标签：⌘ + 数字； ⌘ + 左右方向键
- 切换全屏：⌘ + enter
- 查找：⌘ + f

------

## 分屏

- 垂直分屏：⌘ + d
- 水平分屏：⌘ + ⇧ + d
- 切换屏幕：⌘ + ⌥ + 方向键 ⌘ + [ 或 ⌘ + ]
- 查看历史命令：⌘ + ;

------

## 其他

- 清除当前行：⌃ + U
- 到行首：⌃ + A

- 到行尾：⌃ + E

- 前进后退：⌃ + F / ⌃ + B (相当于左右方向键)

- 上一条命令：⌃ + P

- 搜索命令历史：⌃ + R

- 删除当前光标的字符：⌃ + D

- 删除光标之前的字符：⌃ + H

- 删除光标之前的单词：⌃ + W

- 删除到文本末尾：⌃ + K

- 交换光标处文本：⌃ + T

- 清屏1：⌘ + R
- 清屏2：⌃ + L

------

## 自带实用功能/快捷键

- ⌘ + F 所查找的内容会被自动复制

- ⌘ + D 横着分屏 / ⌘ + ⇧ + D 竖着分屏

- ⌘ + R = clear，而且只是换到新一屏，不会像 clear 一样创建一个空屏

- ⌃ + U 清空当前行，无论光标在什么位置

- 输入开头命令后 按 ⌘ + ; 会自动列出输入过的命令
- ⌃ + K 从光标处删至命令行尾 (本来 ⌃U 是删至命令行首，但iterm中是删掉整行)
- ⌃ + W / ⌥ + D 从光标处删至字首/尾
- ⌃ + Y 粘贴至光标后
- ⌃ + R 搜索命令历史
